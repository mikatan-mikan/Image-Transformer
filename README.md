# 用途

・このソフトウェアは画像を変形し、保存するツールです

##　機能

・画像の切り取り、反転等の4点を用いた変換


# 動作確認済み環境

OS : windows 10 HOME version 21H2

Python : Python 3.7.7 64-bit

# 対応

・読み込み

JPG , PNG , ICO , WEBP

・書き込み

JPG , PNG , ICO , PDF , WEBP
